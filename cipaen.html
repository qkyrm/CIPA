ВНИМАНИЕ! Только перечисленные организации в разделе Контакты https://cipaprogram.org/index.php?do=feedback являются ОФИЦИАЛЬНЫМИ представителями Экзаменационной Сети по программе CIPA! Сертификаты по программе CIPA выдаются только при подтверждении сдачи экзаменов у официальных представителей по проведению экзаменов по программе CIPA.
